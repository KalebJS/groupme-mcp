GroupMe MCP is a [Model Context Protocol](https://modelcontextprotocol.io) server wrapping the [GroupMe API v3](https://dev.groupme.com/docs/v3). Connect it to Claude Desktop or any MCP-compatible AI client and give your assistant full access to your GroupMe account.

## Getting started

<Steps>
  <Step title="Get a GroupMe token">
    Visit [dev.groupme.com](https://dev.groupme.com) to get an access token, or
    run the included OAuth helper script.
  </Step>
  <Step title="Add to Claude Desktop">
    Add the server configuration to your `claude_desktop_config.json` with your
    token.
  </Step>
  <Step title="Start chatting">
    Ask your AI assistant to list your groups, read messages, send a reply, or
    create a bot — it's all available instantly.
  </Step>
</Steps>
